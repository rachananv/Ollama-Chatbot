If you are not redirected automatically, click here to open the chatbot on your computer.

Notes:

  1. This will attempt to open http://localhost:5000/ on the visitor's machine.
  2. Make sure Ollama and the web server are running locally before using this link:
    • ollama serve
    • python web_chatbot.py (use your venv python)
  3. If you want the chatbot available publicly, use ngrok and update this file to the ngrok URL instead.
